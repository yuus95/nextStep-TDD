## 야구게임

123/456 -> nothing

123 / 451 -> ball

123/145 -> Strike 

- 요구 사항 작성 하기
    - 1부터 9까지  입력받아야 한다.
    - 3번 입력받은 수 중 같은 숫자는 없다.
    - 입력받은 수가 com숫자와 같은 자리에 있으면 Strike이다.
    - 입력받은 수가 com숫자에 다른자리에 있을 경우 ball이다.
    - 입력받은 수가 없을 경우 nothing을 출력한다.
    - com숫자는 1~9까지 서로 다른 수로 이루어진 3자리의 랜덤 숫자이다.
